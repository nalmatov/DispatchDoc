    <Tabs>

      #### Настройки в программе **ConfigLBPro** <ProgrammIcon url={ConfigLBProURL} />:
        1. В разделе собственные выберите тип соеденения _Static_.
        2. Придумайте маску сети. К примеру 192.168.0.1.
        3. Придумайте IP-адрес. К примеру 192.168.0.101, если это второй _лифтовой блок_, то 192.168.0.102. IP должен быть уникальным!
        4. Сохраните настройки.
      
      #### Настройки в компьютере/ноутбуке:
        1. В настройках сети укажите статический IP-адрес для компьютера, чтобы он находился в одной сети с лифтовым блоком.
          Например:

            - Если у лифтового блока подсеть 192.168.0.1, его IP-адрес будет 192.168.0.101.
            - В таком случае для компьютера можно указать IP-адрес 192.168.0.100.
          
            Это позволит компьютеру и лифтовому блоку "видеть" друг друга и работать вместе.
        2. Также укажите одну подсеть. Например, если у лб подсеть 192.168.0.1 у компьютера/ноутбука укажите ту же подсеть.

      #### Настройки в программе LKDSProCFG <ProgrammIcon url={LKDSProCfg} />:
      В этой программе настраивается сетевая связь между компьютером/ноутбуком и лифтовым блоком.

export function ProgrammIcon(props) {
  const { type='div', url } = props;

  if (!url) throw new Error('The URL must be passed to the props! props:url');

  const changeBlock = useMemo(() => {
    if (type !== 'div' && type !== 'span') {
      throw new Error("ProgrammIcon: Type can be only the div or span!");

      return null
    };
    const children = <img src={url} alt="Programm icon" className="programm__image" />;

    return createElement(type, { props: {}, className: 'programm' }, children);
  }, [props]);

  return <>{changeBlock}</>;
}
